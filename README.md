# advent2024
Solutions for Advent of Code 2024

## Running locally

### Create env

#### Unix

```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

#### Windows

```bash
python -m venv env
.\env\Scripts\activate
pip install -r requirements.txt
```

### Run

```bash
python day1/part1.py
```
